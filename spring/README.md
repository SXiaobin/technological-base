## How to use Annotation @ImportResource in Spring

标签 @ImportResource 可以加载资源文件，并根据资源文件的配置，使用资源文件中的参数构造 Spring Bean。通过这种方式，我们可以将某个Spring Component 具体实例化几个对象以及每个对象初始参数为何的权柄交由具体项目自有配置。
